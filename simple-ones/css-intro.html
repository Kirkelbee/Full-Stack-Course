Introduction to CSS

  1. Why do we need CSS?

    2. Today there are many ways of adding styling to your website:

    • CSS - every developer must know this one
    • Sass
    • Less

    This was not always the case:

    • Early days : No variation in websites at all.
    • 1997 : W3 proposed HTML 3. This allowed a font tag to define color, size, center etc (all deprecated now as websites got too cluttered).
    • 1996 : CSS had already started

    CSS allows you to style your websites without changing or cluttering the functional parts of the code.

  2. How to add CSS

    3. There are essentially 3 methods

    • Inline - one element on the page with styling placed in the opening tag
    • Internal - one whole page with styling in the head element
    • External - whole website and multiple pages using a CSS style sheet linked in the header of each html page

    Inline

    <p style="property:value " >Element being styled</p>

    For example in this paragraph element, setting the property to 'background' and the value to 'yellow' does this.

    Internal

    By adding a style to the head element of this entire html page I have turned the page background a shade of green.

    <style> html {background : rgb(113, 213, 113);}</style>

    External

    Instead of coding the style in the header you can add to to a styles.css file and link it in the head element

    <link
    rel = "stylesheet"
    href="./styles.css"
    />

    And in the styles.css file you add:
    html {background : rgb(113, 213, 113);}

  3. Selectors

    4. You use a CSS selector to target something you want styled and then assign a property and a value (say background: blue).

    • Element selectors targets the html element tags:
      • h1 - all h1 headings
      • a - all the anchor tags
      • p - all the paragraph tags

    • Class selector: .user-class
      For example....<h2 class="user-class">An h2 element with an assigned class gets styled</h2>

    • ID selector (one unique element per file): #main
      For example....<h2 id="main">An h2 element with ID=main class gets styled</h2>

    • Attribute selectors (id, class, draggable, src, href etc): p[draggable] {color:red}
      You can take these a step further by targeting the attribute and the value, say if p[draggable="true"] {color:red}.

    • The universal selector is *. So * {color:red} targets everything and makes the text red.

  4. Colours

    5. Set the property and the value. For example:

    background-color: blue (this is the background / page colour)

    color: blue (this is the text colour)

    Check out Angela's links to MDN for 'named-color' and the Color Hunt Website.

    You can use named colours (eg: red), or decimal RGB (255, 0, 0) or hexadecimal (#FF0000) and they are all the same.
